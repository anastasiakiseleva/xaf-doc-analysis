---
uid: DevExpress.ExpressApp.DC.IMemberInfo.GetOwnerInstance(System.Object)
name: GetOwnerInstance(Object)
type: Method
summary: Returns an object referenced by a specific object, containing the last property defined in the [IMemberInfo.BindingName](xref:DevExpress.ExpressApp.DC.IMemberInfo.BindingName) property descriptor.
syntax:
  content: object GetOwnerInstance(object obj)
  parameters:
  - id: obj
    type: System.Object
    description: An object of the type which contains the member represented by the [IMemberInfo.BindingName](xref:DevExpress.ExpressApp.DC.IMemberInfo.BindingName) property descriptor.
  return:
    type: System.Object
    description: An object referenced by the object passed as the **obj** parameter, containing the last property defined in the [IMemberInfo.BindingName](xref:DevExpress.ExpressApp.DC.IMemberInfo.BindingName) property descriptor.
