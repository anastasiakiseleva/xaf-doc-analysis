This interface is a part of the [Application Model infrastructure](xref:112580). You do not need to implement this interface in most cases. This interface is used by the **SystemWindowsFormsModule** module to extend the **Options** node (see [](xref:DevExpress.ExpressApp.Model.IModelOptions)) with the properties required for the **MdiShowViewStrategy**.
