name: StartupWindowLoad
type: Event
summary: Occurs before the main [](xref:DevExpress.ExpressApp.Window)'s form is displayed for the first time.
syntax:
  content: public event EventHandler StartupWindowLoad
seealso: []
---
This event is raised when the [Form.Load](https://learn.microsoft.com/en-us/dotnet/api/system.windows.forms.form.load) event of the main Window [WinWindow.Form](xref:DevExpress.ExpressApp.Win.WinWindow.Form) occurs. Handle this event to be notified when the main Window form is loaded.
