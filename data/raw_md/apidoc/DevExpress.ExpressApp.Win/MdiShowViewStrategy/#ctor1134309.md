The **MdiShowViewStrategy** can be used in **XAF** Windows Forms applications as an alternative to the [](xref:DevExpress.ExpressApp.Win.ShowInMultipleWindowsStrategy) that is used by default. Refer to the [](xref:DevExpress.ExpressApp.Win.MdiShowViewStrategy) class description for additional information.

Generally, you don't have to create a Show View Strategy manually. It's created automatically by the [](xref:DevExpress.ExpressApp.XafApplication). When the Application creates an MdiShowViewStrategy, it passes the value of the [IModelOptionsWin.UIType](xref:DevExpress.ExpressApp.Win.SystemModule.IModelOptionsWin.UIType) property as the _mdiMode_ parameter. If you need to create an MdiShowViewStrategy manually, and then assign it to the [WinApplication.ShowViewStrategy](xref:DevExpress.ExpressApp.Win.WinApplication.ShowViewStrategy), get the **IModelOptionsWin.UIType** property value. Alternatively, pass the required MDIMode value.
