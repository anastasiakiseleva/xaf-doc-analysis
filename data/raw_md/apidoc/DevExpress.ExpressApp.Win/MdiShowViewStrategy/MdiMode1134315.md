The **MdiShowViewStrategy** has two modes of operation. When the **MdiMode** property is set to [MdiMode.Standard](xref:DevExpress.ExpressApp.Win.Templates.MdiMode.Standard), [Views](xref:112611) are invoked in separate Windows:

![MdiModeStandard](~/images/mdimodestandard116705.png)

When the **MdiMode** property is set to [MdiMode.Tabbed](xref:DevExpress.ExpressApp.Win.Templates.MdiMode.Tabbed), [Views](xref:112611) are invoked in separate tabs:

![MdiModeTabbed](~/images/mdimodetabbed116706.png)

To set the required mode, use the [IModelOptionsWin.UIType](xref:DevExpress.ExpressApp.Win.SystemModule.IModelOptionsWin.UIType) property of the **Options** node in the Model Editor.
