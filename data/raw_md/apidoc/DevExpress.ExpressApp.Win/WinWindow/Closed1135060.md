name: Closed
type: Event
summary: Occurs when the [](xref:DevExpress.ExpressApp.Win.WinWindow)'s [WinWindow.Form](xref:DevExpress.ExpressApp.Win.WinWindow.Form) is closed.
syntax:
  content: public event EventHandler Closed
seealso:
- linkId: DevExpress.ExpressApp.Win.WinWindow.Closing

To prevent the form from closing, handle the [WinWindow.Closing](xref:DevExpress.ExpressApp.Win.WinWindow.Closing) event and set the **Cancel** argument passed into the event handler to **true**.
