name: Closing
type: Event
summary: Occurs when the [](xref:DevExpress.ExpressApp.Win.WinWindow)'s [WinWindow.Form](xref:DevExpress.ExpressApp.Win.WinWindow.Form) is about to be closed.
syntax:
  content: public event CancelEventHandler Closing
seealso:

Handle this event to prevent the form from closing. For this purpose, set the **Cancel** argument passed into the event handler to **true**.
