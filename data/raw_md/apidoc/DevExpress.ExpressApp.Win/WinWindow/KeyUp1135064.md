name: KeyUp
type: Event
summary: Occurs when a key is released while the [](xref:DevExpress.ExpressApp.Win.WinWindow)'s [WinWindow.Form](xref:DevExpress.ExpressApp.Win.WinWindow.Form) has focus.
syntax:
  content: public event KeyEventHandler KeyUp

Handle this event to perform specific actions in response to a key release. This can be used, for instance, to perform any cleanup necessary after the [WinWindow.KeyDown](xref:DevExpress.ExpressApp.Win.WinWindow.KeyDown) event handler has been executed.
