The **StaticTextViewItem** uses an instance of the **DevExpress.XtraEditors.**[](xref:DevExpress.XtraEditors.LabelControl) to display captions in a UI.

The following image illustrates the **StaticTextViewItem** used to display the "Welcome! Please enter your user name and password below." message:

![Tutorial_SS_Lesson1_1](~/images/tutorial_ss_lesson1_1115528.png)

For general information on Static Text View Items, refer to the [](xref:DevExpress.ExpressApp.Editors.StaticText) class description.

To learn how to add a Static Text View Item to a Detail View, refer to the [View Items](xref:112612) topic.
