The **StaticImageViewItem** uses an instance of the **PictureEdit** control to display images in a UI. This control is a **System.Windows.Forms.PictureBox** descendant, custom tailored for XAF specifics.

The following picture illustrates the **StaticImageViewItem** used to display report pictures on the left side of a [Detail View](xref:112611):

![StaticImageDetailItem_Win](~/images/staticimagedetailitem_win116334.png)

For general information on Static Image View Items, refer to the [](xref:DevExpress.ExpressApp.Editors.StaticImage) class description.

To learn how to add a Static Image View Item to a Detail View, refer to the [View Items](xref:112612) topic.
