The **WinActionContainerViewItem** uses an instance of the **DevExpress.ExpressApp.Win.Templates.ActionContainers.ButtonsContainer** to display [Actions](xref:112622) in a UI.

The following picture illustrates the **WinActionContainerViewItem** used to display the "Register" and "Get Key By Email" [](xref:DevExpress.ExpressApp.Actions.SimpleAction)s:

![ActionInDetailViewWin](~/images/actionindetailviewwin116710.png)

For general information on Action Container View Items, refer to the [](xref:DevExpress.ExpressApp.Editors.ActionContainerViewItem) class description.

To learn how to add an Action Container View Item to a Detail View, refer to the [View Items](xref:112612) and [How to: Include an Action to a Detail View Layout](xref:112816) topics.
