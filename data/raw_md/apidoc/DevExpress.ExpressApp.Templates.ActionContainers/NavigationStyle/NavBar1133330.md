name: NavBar
type: Field
summary: Specifies that a navbar control is used as a navigation control. In this instance, the first-level child nodes of the [](xref:DevExpress.ExpressApp.SystemModule.IModelRootNavigationItems) node correspond to the navbar control's groups. These groups cannot be used to navigate to [Views](xref:112611). The groups only serve as containers for other child nodes. In XAF Blazor applications, the `NavBar` option is equivalent to the `Accordion` option.
syntax:
  content: NavBar = 1
