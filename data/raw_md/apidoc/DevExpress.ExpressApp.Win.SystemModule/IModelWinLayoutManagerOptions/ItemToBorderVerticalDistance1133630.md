uid: DevExpress.ExpressApp.Win.SystemModule.IModelWinLayoutManagerOptions.ItemToBorderVerticalDistance
name: ItemToBorderVerticalDistance
type: Property
summary: |-
  Specifies the vertical indents between the following layout elements:


  1. A Layout Group's horizontal borders and the Layout Items contained in the Group. Has no effect on Layout Groups with hidden text.


  2. A Layout Group's horizontal borders and the Detail View's edges.
syntax:
  content: |-
    [DefaultValue(10)]
    int ItemToBorderVerticalDistance { get; set; }
  parameters: []
  return:
    type: System.Int32
    description: An integer value specifying vertical indents between the layout elements.
seealso:
- linkId: "112817"
---
This property is in effect for **XAF** Windows Forms applications only.
