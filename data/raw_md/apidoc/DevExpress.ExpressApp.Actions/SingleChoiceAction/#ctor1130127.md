  parameters:
  - id: owner
    type: DevExpress.ExpressApp.Controller

  - id: id

    type: System.String

Via this method, a Single Choice Action is created and added to the Controller specified by the _owner_ parameter.

Usually, you do not need to create [](xref:DevExpress.ExpressApp.Actions.SingleChoiceAction) objects manually. Instead, use the parent Controller's **Designer** to add Actions. For details, refer to the [Actions](xref:112622) topic.
