Currently, members of the **IContainer** interface are implemented in the [](xref:DevExpress.ExpressApp.Controller) class only. So, the _contianer_  parameter is used to initialize the [ActionBase.Controller](xref:DevExpress.ExpressApp.Actions.ActionBase.Controller) property of the created Parametrized Action.

Usually, you do not need to create [](xref:DevExpress.ExpressApp.Actions.ParametrizedAction) objects manually. Instead, use the parent Controller's Designer to add Actions. For details, refer to the [Actions](xref:112622) topic.
