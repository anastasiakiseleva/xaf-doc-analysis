name: Cancel
type: Event
summary: Occurs when an end-user clicks the cancelling button on the Pop-up Window Show Action's pop-up Window.
syntax:
  content: public event EventHandler Cancel
seealso: []
---
Handle this event to execute custom code when clicking the cancel button on the current Pop-up Window Show Action's pop-up Window.

To execute custom code when clicking the accept button, handle the [PopupWindowShowAction.Execute](xref:DevExpress.ExpressApp.Actions.PopupWindowShowAction.Execute) event.
