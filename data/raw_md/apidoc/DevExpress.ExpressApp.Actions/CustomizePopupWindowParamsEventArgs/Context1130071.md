All windows are created in a particular context; for example, the main [](xref:DevExpress.ExpressApp.Window) is created in the [TemplateContext.ApplicationWindow](xref:DevExpress.ExpressApp.TemplateContext.ApplicationWindow) context. In Windows Forms applications, a [Template](xref:112609) to be set to a Window is determined via the context. Use this property to get or set a context for the pop-up Window which is created for the parent [CustomizePopupWindowParamsEventArgs.Action](xref:DevExpress.ExpressApp.Actions.CustomizePopupWindowParamsEventArgs.Action).

By default, the **Context** property is set to the [TemplateContext.PopupWindow](xref:DevExpress.ExpressApp.TemplateContext.PopupWindow) context when a Detail View is set for the created pop-up Window, and the [TemplateContext.LookupWindow](xref:DevExpress.ExpressApp.TemplateContext.LookupWindow) context when a List View is set for the pop-up Window.
