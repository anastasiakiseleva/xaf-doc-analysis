---
uid: DevExpress.ExpressApp.Actions.ActionBase.Changed
name: Changed
type: Event
summary: Occurs after one of the current ActionBase object's properties has been changed.
syntax:
  content: |-
    [Browsable(false)]
    public event EventHandler<ActionChangedEventArgs> Changed
seealso: []
---
Use the event handler's **ActionChangedEventArgs.ChangedPropertyType** parameter to indicate what property has been changed. This parameter can have the following values:

| Parameter Value | Property Changed |
|---|---|
| **Caption** | [ActionBase.Caption](xref:DevExpress.ExpressApp.Actions.ActionBase.Caption) or [ParametrizedAction.ShortCaption](xref:DevExpress.ExpressApp.Actions.ParametrizedAction.ShortCaption) |
| **Image** | [ActionBase.ImageName](xref:DevExpress.ExpressApp.Actions.ActionBase.ImageName) |
| **Enabled** | [ActionBase.Enabled](xref:DevExpress.ExpressApp.Actions.ActionBase.Enabled) |
| **Active** | [ActionBase.Active](xref:DevExpress.ExpressApp.Actions.ActionBase.Active) |
| **ToolTip** | [ActionBase.ToolTip](xref:DevExpress.ExpressApp.Actions.ActionBase.ToolTip) |
| **ConfirmationMessage** | [ActionBase.ConfirmationMessage](xref:DevExpress.ExpressApp.Actions.ActionBase.ConfirmationMessage) |
| **TargetObjectsCriteria** | [ActionBase.TargetObjectsCriteria](xref:DevExpress.ExpressApp.Actions.ActionBase.TargetObjectsCriteria) |
| **TargetObjectsCriteriaMode** | [ActionBase.TargetObjectsCriteria](xref:DevExpress.ExpressApp.Actions.ActionBase.TargetObjectsCriteria) |
