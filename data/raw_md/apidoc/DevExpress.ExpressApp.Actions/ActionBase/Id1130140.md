By default, this property is set to a GUID. You can set another value in the Controller's contructor or **Designer**. This value will be saved to the [Application Model](xref:112580)'s [](xref:DevExpress.ExpressApp.Model.IModelAction) node.

> [!NOTE]
> An Action's identifier must be unique. For instance, you can set values like namespaces: "MyApp.FinancialOperations.AddTurnoverAction".
