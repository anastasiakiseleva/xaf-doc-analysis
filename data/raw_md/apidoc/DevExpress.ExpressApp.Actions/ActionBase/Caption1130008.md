By default, this property is set to the [ActionBase.Id](xref:DevExpress.ExpressApp.Actions.ActionBase.Id) property value. You can set another value in the Controller's constructor. This value will be saved to the [Application Model](xref:112580)'s [!include[Node_Action](~/templates/node_action111373.md)] node. You can change this value via the [Model Editor](xref:112830). In a UI, the value which is specified in the Application Model's .xafml file, loaded last, will be displayed. For information on the order of Application Model differences loading, refer to the [Application Model Basics](xref:112580) topic.

When setting a value to this property, the [View.CaptionChanged](xref:DevExpress.ExpressApp.View.CaptionChanged) event is raised.
