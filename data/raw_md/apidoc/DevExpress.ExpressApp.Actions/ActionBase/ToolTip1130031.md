Use the **ToolTip** property to specify a custom text for an Action's tooltip. The tooltip is displayed when a user hovers a mouse over an Action.

By default, a tooltip is set to the [ActionBase.Caption](xref:DevExpress.ExpressApp.Actions.ActionBase.Caption) property value. You can set another value in the Controller's constructor or **Designer**. This value will be saved to the [Application Model](xref:112580)'s [](xref:DevExpress.ExpressApp.Model.IModelAction) node. You can change this value in the [Model Editor](xref:112830). 

When the property's value changes, the [ActionBase.Changed](xref:DevExpress.ExpressApp.Actions.ActionBase.Changed) event is raised.
