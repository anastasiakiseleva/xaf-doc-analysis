  content: public enum ActionMeaning

Members of the **ActionMeaning** enumeration specify an Action control's type. To get the enumeration value which is currently used by an Action, use the [ActionBase.ActionMeaning](xref:DevExpress.ExpressApp.Actions.ActionBase.ActionMeaning) property.
