Use this constructor if you have an Application Model Node that defines the created Choice Action Item. By default, the [!include[Node_Action](~/templates/node_action111373.md)] | **ChoiceActionItems** | **ChoiceActionItem** node is used to store information on Choice Action Items.

The specified Application Model node will be used to initialize the [ChoiceActionItem.Id](xref:DevExpress.ExpressApp.Actions.ChoiceActionItem.Id), [ChoiceActionItem.Caption](xref:DevExpress.ExpressApp.Actions.ChoiceActionItem.Caption) and [ChoiceActionItem.ImageName](xref:DevExpress.ExpressApp.Actions.ChoiceActionItem.ImageName) properties.

If you do not need to associate the created Item with an object, use the constructor with the single _info_ parameter.
