Use this method to create an [](xref:DevExpress.ExpressApp.Actions.ActionUrl) object and add it to the [Controller](xref:112621) specified by the _container_ parameter. Generally, you can add the new Action to another object that implements the **IContainer** interface. However, the **XAF** provides the only class that contains members of the **IContainer** interface - the [](xref:DevExpress.ExpressApp.Controller) class.

The Action created via this method will be mapped to the [Action Container](xref:112610) specified as default in the current [Template](xref:112609), because its [ActionBase.Category](xref:DevExpress.ExpressApp.Actions.ActionBase.Category) property is set to the "Unspecified" value.

If you need to create an ActionUrl, add it to a specified Controller, and initialize its properties, use another ActionUrl constructor.
