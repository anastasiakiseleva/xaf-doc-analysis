uid: DevExpress.ExpressApp.Actions.ActionUrl.#ctor(DevExpress.ExpressApp.Controller,System.String,System.String)
name: ActionUrl(Controller, String, String)
type: Constructor
summary: Creates an [](xref:DevExpress.ExpressApp.Actions.ActionUrl) and initializes its properties by specified values.
syntax:
  content: public ActionUrl(Controller owner, string id, string category)
  parameters:
  - id: owner
    type: DevExpress.ExpressApp.Controller
    description: A [](xref:DevExpress.ExpressApp.Controller) object that represents a [Controller](xref:112621) to which the created Action will be added. This object is assigned to the [ActionBase.Controller](xref:DevExpress.ExpressApp.Actions.ActionBase.Controller) property.
  - id: id
    type: System.String
    description: A String value representing the ID of the created Action. This value is assigned to the [ActionBase.Id](xref:DevExpress.ExpressApp.Actions.ActionBase.Id) property.
  - id: category
    type: System.String
    description: A String value that represents a category of the created Action. This value is assigned to the [ActionBase.Category](xref:DevExpress.ExpressApp.Actions.ActionBase.Category) property, and is used to map this Action to the appropriate [Action Container](xref:112610).
seealso: []
---
Use this method to create an [](xref:DevExpress.ExpressApp.Actions.ActionUrl) object, add it to the Controller specified by the _owner_ parameter and initialize its [ActionBase.Id](xref:DevExpress.ExpressApp.Actions.ActionBase.Id) and [ActionBase.Category](xref:DevExpress.ExpressApp.Actions.ActionBase.Category) properties by the passed values. For the _category_ parameter, use one of the **DevExpress.Persistent.Base.PredefinedCategory** enumeration values.
