  content: public SimpleAction(Controller owner, string id, string category, SimpleActionExecuteEventHandler execute)
  parameters:
  - id: owner
    type: DevExpress.ExpressApp.Controller
    description: A [](xref:DevExpress.ExpressApp.Controller) object that represents a [Controller](xref:112621) to which the created SimpleAction is added. This object is assigned to the [ActionBase.Controller](xref:DevExpress.ExpressApp.Actions.ActionBase.Controller) property.
  - id: id
    type: System.String
    description: A string value representing the created Simple's identifier. This value is assigned to the [ActionBase.Id](xref:DevExpress.ExpressApp.Actions.ActionBase.Id) property.
  - id: category
    type: System.String
    description: A string value that represents a category of the created Simple Action. This value is assigned to the [ActionBase.Category](xref:DevExpress.ExpressApp.Actions.ActionBase.Category) property and used to map this Action to the appropriate [Action Container](xref:112610).
  - id: execute
    type: DevExpress.ExpressApp.Actions.SimpleActionExecuteEventHandler
    description: A [](xref:DevExpress.ExpressApp.Actions.SimpleActionExecuteEventHandler) object that represents a handler for the [SimpleAction.Execute](xref:DevExpress.ExpressApp.Actions.SimpleAction.Execute) event of the created Simple Action.

This method is used to create [](xref:DevExpress.ExpressApp.Actions.SimpleAction) objects at runtime. Via this method, a new [](xref:DevExpress.ExpressApp.Actions.SimpleAction) object is created, added to the specified Controller and initialized. In addition, the [SimpleAction.Execute](xref:DevExpress.ExpressApp.Actions.SimpleAction.Execute) event is handled by the handler passed as the _Execute_ parameter.
