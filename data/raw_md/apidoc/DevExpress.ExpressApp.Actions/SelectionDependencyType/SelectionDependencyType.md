Members of the [](xref:DevExpress.ExpressApp.Actions.SelectionDependencyType) enumeration specify all available object selection types. To get the enumeration value which is currently used by an Action, use the [ActionBase.SelectionDependencyType](xref:DevExpress.ExpressApp.Actions.ActionBase.SelectionDependencyType) property.
