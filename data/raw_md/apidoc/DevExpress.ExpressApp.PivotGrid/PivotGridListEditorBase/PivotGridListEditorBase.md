[List Views](xref:112611) use List Editors to display object collections in a UI. Pivot Grid List Editors provide the capability to analyze data via pivot tables in an efficient manner. `PivotGridListEditorBase` is the abstract base class from which the WinForms-specific built-in [](xref:DevExpress.ExpressApp.PivotGrid.Win.PivotGridListEditor) derives.
