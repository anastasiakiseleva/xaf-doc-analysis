  content: |-
    [DefaultValue(false)]
    bool ShowColumnTotals { get; set; }

To display the chart together with the pivot table, set the [IPivotSettings.ShowChart](xref:DevExpress.ExpressApp.PivotGrid.IPivotSettings.ShowChart) property to **true**.
