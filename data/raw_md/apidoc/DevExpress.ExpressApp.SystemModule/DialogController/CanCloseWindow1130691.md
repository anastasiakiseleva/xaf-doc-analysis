name: CanCloseWindow
type: Property
summary: Specifies whether to close the current pop-up Window when executing the Dialog Controller's [DialogController.AcceptAction](xref:DevExpress.ExpressApp.SystemModule.DialogController.AcceptAction) or [DialogController.CancelAction](xref:DevExpress.ExpressApp.SystemModule.DialogController.CancelAction).
syntax:
  content: public bool CanCloseWindow { get; set; }
  parameters: []
  return:
    type: System.Boolean
    description: "**true** if the current pop-up Window is closed after using the Dialog Controller's [DialogController.AcceptAction](xref:DevExpress.ExpressApp.SystemModule.DialogController.AcceptAction) or [DialogController.CancelAction](xref:DevExpress.ExpressApp.SystemModule.DialogController.CancelAction); otherwise, **false**."
seealso: []
---
By default, the **CanCloseWindow** property is set to **true**.
