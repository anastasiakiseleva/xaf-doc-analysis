The **CustomizeNotificationViewParameters** event occurs before the [notification](xref:113690) window is shown. Handle the **CustomizeShowViewParametersEventArgs** event to customize the notification view parameters which are passed as the event _showViewParameters_ argument of the [](xref:DevExpress.ExpressApp.ShowViewParameters) type.
