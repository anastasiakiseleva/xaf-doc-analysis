This method sets the values passed as parameters for the corresponding private fields of the [](xref:DevExpress.ExpressApp.SystemModule.AboutInfo) instance. This field values are then returned by the corresponding properties, if these properties have not been set directly in code.

This method is called by the other **Init** method overloads, which pass values of the **Application** node's properties as the corresponding parameters.
