This property value takes place in the [AboutInfo.AboutInfoString](xref:DevExpress.ExpressApp.SystemModule.AboutInfo.AboutInfoString) property's default value construction. [](xref:DevExpress.ExpressApp.Win.SystemModule.AboutInfoController) sets this property to the value of the Application node's `Company` property, before it displays the `AboutInfoString` in a UI. You can change this property value at any time before the `AboutInfoController` and `AboutInfoControl` display the information. In this case, XAF displays your custom value.
