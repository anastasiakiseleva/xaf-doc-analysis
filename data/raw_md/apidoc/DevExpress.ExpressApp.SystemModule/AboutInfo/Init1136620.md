[](xref:DevExpress.ExpressApp.Win.SystemModule.AboutInfoController) calls the `Init` method before showing the about information on the current application. This method initializes the `AboutInfo` object's properties with the values of the corresponding properties from the Application Model's Application node (see [](xref:DevExpress.ExpressApp.Model.IModelApplication)). However, if you set the `AboutInfo` object's properties in code, XAF does not use the values that this method specifies.

Use this method if you implement a custom Controller or control to show information on the current application in a custom manner.
