Call this method to immediate recreate all navigation items defined in the [Application Model](xref:112580). Refer to the following example for more information: [How to show the number of List View items in the Navigation Control](https://supportcenter.devexpress.com/ticket/details/T1002750/xaf-how-to-show-the-number-of-list-view-items-in-the-navigation-control).
