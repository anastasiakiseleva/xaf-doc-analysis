  content: public event EventHandler<NavigationItemCreatedEventArgs> NavigationItemCreated

Handle this event to perform custom actions when navigation items are created in the navigation control. Primarily, this event is used when implementing custom context navigation (see [Navigation System](xref:113198)). To see an example of handling the **NavigationItemCreated** event, refer to the [How to: Implement Custom Context Navigation](xref:113200) topic.
