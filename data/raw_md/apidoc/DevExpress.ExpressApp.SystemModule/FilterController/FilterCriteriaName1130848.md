---
uid: DevExpress.ExpressApp.SystemModule.FilterController.FilterCriteriaName
name: FilterCriteriaName
type: Field
summary: Returns the name of the item in the Criteria collection of the current List View's CollectionSource. This item is updated each time a new filter is selected via the [](xref:DevExpress.ExpressApp.SystemModule.FilterController)'s **Filter** Action. By default, this attribute is called "ListViewFilter".
syntax:
  content: public const string FilterCriteriaName = "ListViewFilter"
  return:
    type: System.String
    description: The filter criterion name. (A List View stores Filter Criteria in its CollectionSource.)
seealso: []
