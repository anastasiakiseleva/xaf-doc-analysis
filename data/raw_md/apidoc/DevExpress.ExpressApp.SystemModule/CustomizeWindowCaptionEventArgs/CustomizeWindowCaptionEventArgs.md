The **CustomizeWindowCaptionEventArgs** class declares the [CustomizeWindowCaptionEventArgs.WindowCaption](xref:DevExpress.ExpressApp.SystemModule.CustomizeWindowCaptionEventArgs.WindowCaption) property specific to the [WindowTemplateController.CustomizeWindowCaption](xref:DevExpress.ExpressApp.SystemModule.WindowTemplateController.CustomizeWindowCaption) event. This event is raised before the [Window](xref:112608) caption is updated. Handle the **CustomizeWindowCaption** event in a custom [Window Controller](xref:112621) to modify a Window caption. Refer to the [How to: Customize a Window Caption](xref:113252) for details.
