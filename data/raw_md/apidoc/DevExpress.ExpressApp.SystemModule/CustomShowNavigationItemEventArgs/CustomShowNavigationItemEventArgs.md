The **CustomShowNavigationItem** event occurs when an end-user clicks a navigation item in the navigation control. You can handle this event to perform custom actions, such as displaying custom [Views](xref:112611). To see an example of handling the [ShowNavigationItemController.CustomShowNavigationItem](xref:DevExpress.ExpressApp.SystemModule.ShowNavigationItemController.CustomShowNavigationItem) event, refer to the [How to: Implement Custom Context Navigation](xref:113200) topic.
