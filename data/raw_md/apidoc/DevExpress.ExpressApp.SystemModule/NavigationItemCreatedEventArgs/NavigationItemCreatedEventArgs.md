The **NavigationItemCreated** event occurs after a navigation item has been created in the navigation control. You can handle this event to implement custom context navigation. To see an example of handling the [ShowNavigationItemController.NavigationItemCreated](xref:DevExpress.ExpressApp.SystemModule.ShowNavigationItemController.NavigationItemCreated) event, refer to the [How to: Implement Custom Context Navigation](xref:113200) topic.
