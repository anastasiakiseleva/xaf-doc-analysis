name: Size
type: Property
summary: Specifies the maximum number of characters that users can type in Property Editors of the current string property. You can also specify the current member's [XPMemberInfo.MappingFieldSize](xref:DevExpress.Xpo.Metadata.XPMemberInfo.MappingFieldSize) property (XPO only) or apply [](xref:DevExpress.ExpressApp.DC.FieldSizeAttribute) to it. Property Editors ignore the **IModelMember.Size** property if its value is more than the [](xref:DevExpress.Xpo.Metadata.XPMemberInfo.MappingFieldSize) or [](xref:DevExpress.ExpressApp.DC.FieldSizeAttribute) value.
syntax:
  content: int Size { get; set; }
  parameters: []
  return:
    type: System.Int32
    description: The maximum number of characters that users can type in Property Editors of the current string property.
seealso:
- linkId: DevExpress.ExpressApp.Model.IModelMemberViewItem.MaxLength
---
This property is initialized with a value from the [IMemberInfo.Size](xref:DevExpress.ExpressApp.DC.IMemberInfo.Size) property.

To specify the maximum number of characters that users can enter into the specific Property Editor, use the [IModelMemberViewItem.MaxLength](xref:DevExpress.ExpressApp.Model.IModelMemberViewItem.MaxLength) property.
