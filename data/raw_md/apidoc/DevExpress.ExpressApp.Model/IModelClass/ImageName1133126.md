By default, this property is set to the [](xref:DevExpress.Persistent.BaseImpl.BaseObject) node's `ImageName` property value.

In the Model Editor, you can select an image via the **Image Picker** dialog. To invoke this dialog, click the ellipsis button (![EllipsisButton](~/images/ellipsisbutton116182.png)) to the right of the `ImageName` value.
