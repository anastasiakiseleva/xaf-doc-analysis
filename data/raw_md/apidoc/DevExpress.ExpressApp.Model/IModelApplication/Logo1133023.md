The image should be added as an embedded resource (see <xref:112792>). In WinForms applications, the specified image is displayed in the window invoked by the [AboutInfoController.AboutInfoAction](xref:DevExpress.ExpressApp.Win.SystemModule.AboutInfoController.AboutInfoAction) Action.

![LogoImage](~/images/logoimage116232.png)

An example of using this property is provided in the [Change an Application Logo and Info](xref:113156) topic.
