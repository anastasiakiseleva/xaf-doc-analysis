This property value is passed to the [LayoutControlItem.MaxSize](xref:DevExpress.XtraLayout.LayoutControlItem.MaxSize) property. If using the **MaxSize** property produces unexpected effects, use the [Control.MaximumSize](https://learn.microsoft.com/en-us/dotnet/api/system.windows.forms.control.maximumsize) property as shown below.

# [C#](#tab/tabid-csharp)

```csharp
using System;
using DevExpress.ExpressApp;
using DevExpress.ExpressApp.Editors;
using System.Windows.Forms;
//...
public class MyController : ViewController <DetailView> {
    //...
    protected override void OnActivated() {
        base.OnActivated();
        ViewItem item = View.FindItem("FullName");
        item.ControlCreated += Item_ControlCreated;
    }
    private void Item_ControlCreated(object sender, EventArgs e) {
        ((ViewItem)sender).ControlCreated -= Item_ControlCreated;
        ((Control)((ViewItem)sender).Control).MaximumSize = new System.Drawing.Size(300, 0);
    }
}
```
