This property value is passed to the [LayoutControlItem.MinSize](xref:DevExpress.XtraLayout.LayoutControlItem.MinSize) property. If using the **MinSize** property produces unexpected effects, use the [Control.MinimumSize](https://learn.microsoft.com/en-us/dotnet/api/system.windows.forms.control.minimumsize#System_Windows_Forms_Control_MinimumSize) property as shown below.

# [C#](#tab/tabid-csharp)

```csharp
using System;
using DevExpress.ExpressApp;
using DevExpress.ExpressApp.Editors;
using System.Windows.Forms;
//...
public class MyController : ViewController <DetailView> {
    //...
    protected override void OnActivated() {
        base.OnActivated();
        ViewItem item = View.FindItem("FullName");
        item.ControlCreated += Item_ControlCreated;
    }
    private void Item_ControlCreated(object sender, EventArgs e) {
        ((ViewItem)sender).ControlCreated -= Item_ControlCreated;
        ((Control)((ViewItem)sender).Control).MinimumSize = new System.Drawing.Size(300, 0);
    }
}
```
