This interface is a part of the [Application Model infrastructure]( xref:112580). You do not need to implement this interface in most cases.

The **IModelStaticImage** interface allows end-users to access and specify [StaticImage](xref:DevExpress.ExpressApp.Editors.StaticImage) settings in the [Model Editor](xref:112582). 

If you need to modify **StaticImage** objects in code, access **StaticImage** members and events directly.
