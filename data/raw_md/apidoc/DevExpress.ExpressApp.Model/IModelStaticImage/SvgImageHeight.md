This property specifies the height the current [Static Image](xref:DevExpress.ExpressApp.Editors.StaticImage) uses to render an SVG image.

To use this property, set the [ImageSizeMode](xref:DevExpress.Persistent.Base.ImageSizeMode) to `AutoSize`, `Normal`, or `CenterImage` in the [Model Editor](xref:112582).
