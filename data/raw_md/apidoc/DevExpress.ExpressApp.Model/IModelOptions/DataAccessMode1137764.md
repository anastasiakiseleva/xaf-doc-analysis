This property value is used as a default value for [IModelListView.DataAccessMode](xref:DevExpress.ExpressApp.Model.IModelListView.DataAccessMode).

[!include[DataAccessModes_Links](~/templates/dataaccessmodes_links111899.md)]

When the **DataAccessMode** property is changed, all List Views, _except for the autogenerated nested List Views_, have their **DataAccessMode** property changed as well.

> [!NOTE]
> All manually created List Views, including nested List Views, have the same [IModelListView.DataAccessMode](xref:DevExpress.ExpressApp.Model.IModelListView.DataAccessMode) property value as the global **DataAccessMode** option.
