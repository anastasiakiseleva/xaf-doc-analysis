The **EnableLayoutGroupImages** property allows displaying images for [tabbed layout groups](xref:2170) in a [Detail View](xref:112611#detail-view).

![EnableLayoutGroupImages_DetailView](~/images/tabbed-layout-groups.png) 

Set the **EnableLayoutGroupImages** property to **true** as shown below to enable this option for all Detail Views your application contains:

![EnableLayoutGroupImages_Options](~/images/EnableLayoutGroupImages_Options.png)

The following image demonstrates how to change this option for a particular Detail View:

![EnableLayoutGroupImages_DetailView](~/images/EnableLayoutGroupImages_DetailView.png)
