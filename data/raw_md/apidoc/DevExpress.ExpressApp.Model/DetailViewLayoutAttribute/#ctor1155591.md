  content: public DetailViewLayoutAttribute(string groupId, LayoutGroupType groupType)
  parameters:
  - id: groupId
    type: System.String
    description: A string value specifying the group identifier in which the editor of the target property is placed. If a group with the specified identifier is missing in the default layout, it is created and added to the **Main** group. The type of the created group is specified using the _groupType_ parameter.
  - id: groupType
    type: DevExpress.ExpressApp.Model.LayoutGroupType
    description: A [](xref:DevExpress.ExpressApp.Model.LayoutGroupType) enumeration value specifying the type of a new group which is created if a group with the _groupId_ identifier is missing in the default layout.
seealso: []

If several properties have the same _groupId_ values, their _groupType_ parameter should be the same as well.
