  content: public RuleRangeAttribute(string id, DefaultContexts targetContexts, object minimumValue, object maximumValue, string messageTemplate)
  parameters:
  - id: id
    type: System.String
    description: A string value representing the current rule identifier. You can pass an empty value as the _id_ parameter. An appropriate identifier will begenerated automatically, and set for the **ID** property of the **Validation** | **Rules** | **Rule** node in the [Application Model](xref:112580).
  - id: targetContexts
    type: DevExpress.Persistent.Validation.DefaultContexts
    description: A **DefaultContexts** enumeration value that specifies a context for checking the current rule. This value is set for the [RuleBaseAttribute.TargetContextIDs](xref:DevExpress.Persistent.Validation.RuleBaseAttribute.TargetContextIDs) property.
  - id: minimumValue
    type: System.Object
    description: An object that represents the left end point of the range the target property's value should be within.
  - id: maximumValue
    type: System.Object
    description: An object that represents the right end point of the range the target property's value should be within.
  - id: messageTemplate
    type: System.String
    description: A string value representing information defining the current rule. This value is set for the [RuleBaseAttribute.CustomMessageTemplate](xref:DevExpress.Persistent.Validation.RuleBaseAttribute.CustomMessageTemplate) property.
