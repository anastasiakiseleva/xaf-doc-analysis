Use the `Validator`'s @DevExpress.Persistent.Validation.Validator.GetService(System.IServiceProvider) method to get a @DevExpress.Persistent.Validation.RuleSet object. This object contains validation rules declared in the application. The methods of this object allow you to trigger validation programmatically. For additional information and an example, refer to the following topic: <xref:113010#trigger-validation-programmatically>.
