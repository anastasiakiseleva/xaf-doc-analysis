  content: public RuleValueComparisonAttribute(string id, string targetContextIDs, ValueComparisonType operatorType, object rightOperand)
  parameters:
  - id: id
    type: System.String
    description: A string value representing the current rule identifier. You can pass an empty value as the _id_ parameter. An appropriate identifier will begenerated automatically, and set for the **ID** property of the **Validation** | **Rules** | **Rule** node in the [Application Model](xref:112580).
  - id: targetContextIDs
    type: System.String
    description: A string value that specifies a list of identifiers of the contexts when the current rule should be checked. This value is set for the [RuleBaseAttribute.TargetContextIDs](xref:DevExpress.Persistent.Validation.RuleBaseAttribute.TargetContextIDs) property.
  - id: operatorType
    type: DevExpress.Persistent.Validation.ValueComparisonType
    description: A [](xref:DevExpress.Persistent.Validation.ValueComparisonType) enumeration value that represents the operator type to be used when comparing values.
  - id: rightOperand
    type: System.Object
    description: An object that represents the value to be compared with the target property's value.
