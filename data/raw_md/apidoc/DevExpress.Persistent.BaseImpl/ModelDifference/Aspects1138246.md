By default, the aspect with the empty [ModelDifferenceAspect.Name](xref:DevExpress.Persistent.BaseImpl.ModelDifferenceAspect.Name) value is used. When the application is [localized](xref:113298), model differences from the aspect with the **Name** set to the current [language code](https://learn.microsoft.com/en-us/dotnet/api/system.globalization.cultureinfo) are also used.
