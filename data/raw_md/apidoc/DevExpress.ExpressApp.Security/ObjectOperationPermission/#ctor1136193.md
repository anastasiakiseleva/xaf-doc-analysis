name: ObjectOperationPermission(Type, String, String)

syntax:
  content: public ObjectOperationPermission(Type objectType, string criteria, string operation)
  parameters:
  - id: objectType
    type: System.Type
    description: A **System.Type** object which is the type of the secured object.
  - id: criteria
    type: System.String
    description: A string which specifies a [criteria](xref:4928) satisfied by a the secured object.
  - id: operation
    type: System.String
    description: A string which is the name of the secured operation. Supported operations are "Read", "Write", "Delete" and "Navigate".
seealso: []
---
If the specified operation is not supported, a **System.InvalidOperationException** is thrown.
