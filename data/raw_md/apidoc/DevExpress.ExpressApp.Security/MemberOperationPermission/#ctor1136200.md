  content: public MemberOperationPermission(Type objectType, string memberName, string operation)
  parameters:
  - id: objectType
    type: System.Type
    description: A **System.Type** object which is the type exposing the secured member.
  - id: memberName
    type: System.String
    description: A string which is the name of the secured member.
  - id: operation
    type: System.String
    description: A string which is the name of the secured operation. Supported operations are "Read" and "Write".

If the specified operation is not supported, a **System.InvalidOperationException** is thrown.
