An EasyTest command can have a primary parameter and several secondary parameters. Additionally, an extra parameter can be passed as the primary parameter's argument. This class' properties can be used to access different types of the command's parameters. For the complete description of EasyTest syntax, refer to the [EasyTest Script Reference](xref:113208) topic.
