Use this constructor to create an instance of the [](xref:DevExpress.EasyTest.Framework.CommandException) class when it is required to handle an error in [EasyTest](xref:113211) command implementation (e.g. wrong parameter count). To get the current position in the script, use the [Command.StartPosition](xref:DevExpress.EasyTest.Framework.Command.StartPosition) property. An example is provided in the [How to: Implement a Custom EasyTest Command](xref:113340) topic.
