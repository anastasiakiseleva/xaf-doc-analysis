name: Show(String, Exception)

syntax:
  content: public virtual void Show(string caption, Exception exception)
  parameters:
  - id: caption
    type: System.String
    description: A string, which is the exception dialog form caption.
  - id: exception
    type: System.Exception
    description: A **System.Exception** object specifying an exception to be displayed.
seealso: []

Shows the message box with the "Error" caption, **Exception.Message** text and "OK" button, using the [XtraMessageBox.Show](xref:DevExpress.XtraEditors.XtraMessageBox.Show*) method, and then triggers the [Messaging.ConfirmationDialogClosed](xref:DevExpress.ExpressApp.Win.Core.Messaging.ConfirmationDialogClosed) event, if the exception passed is user friendly (**IUserFriendlyException**). Otherwise, displays the exception dialog form via the **ExceptionDialogForm.ShowMessage** method, with caption passed via the _caption_ parameter. You can override this method in a custom [](xref:DevExpress.ExpressApp.Win.Core.Messaging) descendant.
