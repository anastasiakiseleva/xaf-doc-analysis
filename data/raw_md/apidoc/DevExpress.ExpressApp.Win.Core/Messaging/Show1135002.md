  content: public DialogResult Show(string caption, string message)
  parameters:
  - id: caption

This method uses the [XtraMessageBox.Show](xref:DevExpress.XtraEditors.XtraMessageBox.Show*) method, to display a message box. To change this behavior, override the protected **ShowCore** method in a custom [](xref:DevExpress.ExpressApp.Win.Core.Messaging) descendant (see [How to: Implement a Custom Messaging Class](xref:113312)) topic. The [Messaging.ConfirmationDialogClosed](xref:DevExpress.ExpressApp.Win.Core.Messaging.ConfirmationDialogClosed) event is triggered within this method, after the message box is closed.
