name: Show(Exception)
type: Method
summary: Displays a message box.
syntax:
  content: public void Show(Exception exception)
  parameters:
  - id: exception
    type: System.Exception
    description: A **System.Exception** object representing an exception to be displayed.
seealso: []
---
Shows the message box with the "Error" caption, **Exception.Message** text and "OK" button using the [XtraMessageBox.Show](xref:DevExpress.XtraEditors.XtraMessageBox.Show*) method, and then triggers the [Messaging.ConfirmationDialogClosed](xref:DevExpress.ExpressApp.Win.Core.Messaging.ConfirmationDialogClosed) event, if the exception passed is user friendly (**IUserFriendlyException**). Otherwise, displays the exception dialog form via the **ExceptionDialogForm.ShowMessage** method.
