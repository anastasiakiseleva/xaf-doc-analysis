name: IsCaptionVisible
type: Property
summary: Indicates whether the [StaticText.Caption](xref:DevExpress.ExpressApp.Editors.StaticText.Caption) of the current [View Item](xref:112612) should be visible.
syntax:
  content: public override bool IsCaptionVisible { get; }
  parameters: []
  return:
    type: System.Boolean
    description: "**true** if the current View Item's caption is visible; otherwise, **false**."
seealso: []
---
When the layout manager (either in a Windows Forms or ASP.NET Web Forms application) creates a layout item, it gets this property to determine whether a caption should be displayed for the current View Item.
