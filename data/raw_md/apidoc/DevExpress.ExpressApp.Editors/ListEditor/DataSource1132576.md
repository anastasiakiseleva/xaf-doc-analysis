summary: Specifies a [List Editor](xref:113189)'s data source.
syntax:
  content: public object DataSource { get; set; }
  parameters: []
  return:
    type: System.Object
    description: An **object** which represents the collection containing the objects displayed by the List Editor's control.
seealso: []
---
Use this property to get the required objects from a List Editor's data source.

Do not use this property to manually change the List Editor's data source, because it is managed by the List Editor's View.
