name: IsCaptionVisible
type: Property
summary: Indicates whether the [StaticImage.Caption](xref:DevExpress.ExpressApp.Editors.StaticImage.Caption) of the current [View Item](xref:112612) should be visible.
syntax:
  content: public override bool IsCaptionVisible { get; }
  parameters: []
  return:
    type: System.Boolean
    description: "`true` if the current View Item's caption is visible; otherwise, `false`."
seealso: []
---
When the layout manager creates a layout item, it gets this property to determine whether XAF should display a caption for the current View Item.
