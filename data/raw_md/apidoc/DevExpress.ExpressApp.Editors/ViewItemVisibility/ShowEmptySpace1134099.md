---
uid: DevExpress.ExpressApp.Editors.ViewItemVisibility.ShowEmptySpace
name: ShowEmptySpace
type: Field
summary: Specifies that empty space is displayed instead of UI elements affected by a [conditional appearance rule](xref:113286), when [AppearanceAttribute.AppearanceItemType](xref:DevExpress.ExpressApp.ConditionalAppearance.AppearanceAttribute.AppearanceItemType) is set to `ViewItem` or `LayoutItem`. In XAF ASP.NET Core Blazor applications, this mode functions like the [ViewItemVisibility.Hide](xref:DevExpress.ExpressApp.Editors.ViewItemVisibility.Hide) mode.
syntax:
  content: ShowEmptySpace = 1
seealso: []
---
