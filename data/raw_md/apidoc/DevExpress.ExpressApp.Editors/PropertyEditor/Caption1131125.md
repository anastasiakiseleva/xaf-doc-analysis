By default, this property returns the [IModelCommonMemberViewItem.Caption](xref:DevExpress.ExpressApp.Model.IModelCommonMemberViewItem.Caption) property of the [Application Model](xref:112580)'s node (see [PropertyEditor.Model](xref:DevExpress.ExpressApp.Editors.PropertyEditor.Model)). If this value is an empty string, the [PropertyEditor.PropertyName](xref:DevExpress.ExpressApp.Editors.PropertyEditor.PropertyName) property value is set.

This property can be overridden in the [](xref:DevExpress.ExpressApp.Editors.ViewItem) class' descendants.
