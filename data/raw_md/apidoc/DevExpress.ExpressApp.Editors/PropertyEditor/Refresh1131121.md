type: Method
summary: Refreshes the current Property Editor's control value.
syntax:
  content: public override void Refresh()
seealso: []
---
This method is called when the current Property Editor's Detail View (see [ViewItem.View](xref:DevExpress.ExpressApp.Editors.ViewItem.View)) is refreshed. In addition, it is called when the current object of the View is changed.

This method calls the [PropertyEditor.ReadValue](xref:DevExpress.ExpressApp.Editors.PropertyEditor.ReadValue) method. Note that it can be overridden in the [](xref:DevExpress.ExpressApp.Editors.PropertyEditor) class' descendants.
