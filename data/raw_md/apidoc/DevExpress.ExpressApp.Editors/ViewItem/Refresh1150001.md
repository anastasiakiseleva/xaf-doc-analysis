name: Refresh(Boolean)
type: Method
summary: Refreshes the current View Item. Optionally, refreshes the View Item's data source.
syntax:
  content: public void Refresh(bool refreshDataSource)
  parameters:
  - id: refreshDataSource
    type: System.Boolean
    description: '**true**, if the data source needs to be refreshed as well; otherwise, **false**.'
seealso: []
---
**Refresh** is called when the current Detail View is refreshed.

The **Refresh** method raises an exception if the current View Item is disposed. It can be overridden in descendants.

The **Refresh** method calls [ViewItem.RefreshDataSource](xref:DevExpress.ExpressApp.Editors.ViewItem.RefreshDataSource) when the _refreshDataSource_ parameter is set to true.
