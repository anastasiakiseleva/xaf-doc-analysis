The [File Attachments Module](xref:112781) includes the _DevExpress.ExpressApp.FileAttachment.Win.v<:xx.x:>.dll_ assembly that contains classes for Windows Forms applications only and the `FileAttachmentsWindowsFormsModule` class that inherits from the `ModuleBase` class.

The `FileAttachmentsWindowsFormsModule` class exposes properties that determine how Windows Forms handles file attachments.
