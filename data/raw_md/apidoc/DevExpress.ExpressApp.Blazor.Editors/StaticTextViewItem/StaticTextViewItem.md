The **StaticTextViewItem** uses the [\<label>](https://html.spec.whatwg.org/multipage/forms.html#the-label-element) element to display captions in a UI.

The following image illustrates the **StaticTextViewItem** used to display the "Enter your user name and password below." message:

![StaticTextViewItem_Blazor](~/images/StaticTextViewItem_Blazor.png)

For general information on Static Text View Items, refer to the [](xref:DevExpress.ExpressApp.Editors.StaticText) class description.

To learn how to add a Static Text View Item to a Detail View, refer to the [View Items](xref:112612) topic.
