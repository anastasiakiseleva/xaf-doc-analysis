When you need to create a complex rule that cannot be specified via the [IAppearanceRuleProperties.Criteria](xref:DevExpress.ExpressApp.ConditionalAppearance.IAppearanceRuleProperties.Criteria) property, you can implement a business class method that takes no parameters and returns a Boolean value specifying whether the rule is currently active. In this instance, leave the **Criteria** property unspecified and set the **Method** property to the name of the implemented method.
