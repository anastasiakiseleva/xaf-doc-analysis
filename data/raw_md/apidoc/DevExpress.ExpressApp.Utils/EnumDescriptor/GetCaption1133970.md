syntax:
  content: public string GetCaption(int intValue)
  parameters:
  - id: intValue
    type: System.Int32
    description: An integer value representing the enumeration value index, whose corresponding display caption will be retrieved.
  return:
    type: System.String
    description: A string representing the display caption associated with the specified value of the enumeration represented by the **EnumDescriptor**.

Generally, there is no need to use this **GetCaption** method overload, as there is an overload which takes an enumeration value directly.
