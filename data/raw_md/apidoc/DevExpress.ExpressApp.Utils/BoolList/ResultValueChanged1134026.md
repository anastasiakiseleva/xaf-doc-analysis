name: ResultValueChanged
type: Event
summary: Occurs after the [BoolList.ResultValue](xref:DevExpress.ExpressApp.Utils.BoolList.ResultValue) has changed.
syntax:
  content: public event EventHandler<BoolValueChangedEventArgs> ResultValueChanged
seealso: []
---
Handle this event to be notified when the resulting value of a [](xref:DevExpress.ExpressApp.Utils.BoolList) is changed. If you need to be notified when the **BoolList**'s collection of key/value pairs has been modified, handle the [BoolList.Changed](xref:DevExpress.ExpressApp.Utils.BoolList.Changed) event.
