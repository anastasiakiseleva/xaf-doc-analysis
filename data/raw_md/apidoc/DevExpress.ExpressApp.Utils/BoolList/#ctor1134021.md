This constructor configures the created **BoolList** in the following way:

* The [](xref:DevExpress.ExpressApp.Utils.BoolListOperatorType) mode is set to [BoolListOperatorType.And](xref:DevExpress.ExpressApp.Utils.BoolListOperatorType.And).
* The [](xref:DevExpress.ExpressApp.Utils.ReadByNonExistentKeyMode) mode is set to [ReadByNonExistentKeyMode.ReturnDefault](xref:DevExpress.ExpressApp.Utils.ReadByNonExistentKeyMode.ReturnDefault).
