Handle this event to create a custom [](xref:DevExpress.XtraEditors.XtraForm) with the [](xref:DevExpress.DashboardWin.DashboardDesigner). Create and assign a custom form to the [CreateCustomFormEventArgs.Form](xref:DevExpress.ExpressApp.Dashboards.Win.CreateCustomFormEventArgs.Form) event argument. An example is provided in the [How to: Show a Custom Form with the WinForms Dashboard Designer](xref:118270) topic.
