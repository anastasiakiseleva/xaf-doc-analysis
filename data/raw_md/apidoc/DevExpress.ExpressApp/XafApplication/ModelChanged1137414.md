---
uid: DevExpress.ExpressApp.XafApplication.ModelChanged
name: ModelChanged
type: Event
summary: Occurs when the Application Model instance returned by the [XafApplication.Model](xref:DevExpress.ExpressApp.XafApplication.Model) property is changed.
syntax:
  content: public event EventHandler ModelChanged
seealso: []
---
Subscribe to this event to be notified when the Application Model is changed. For instance, this event is triggered when the Application Model is loaded for the first time and when the used differences are loaded or unloaded.
