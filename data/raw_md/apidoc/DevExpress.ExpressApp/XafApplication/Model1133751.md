You can use this property to access a particular Application Model node (see [Access the Application Model in Code](xref:112810)).

The Application Model object returned by the **Model** property can be changed during the application life cycle (e.g., when the Application Model is loaded for the first time and when the user differences are loaded or unloaded). To be notified of this change, subscribe to the [XafApplication.ModelChanged](xref:DevExpress.ExpressApp.XafApplication.ModelChanged) event.
