This method calls the corresponding methods of the [](xref:DevExpress.ExpressApp.XafApplication) class descendants. In the **WinApplication** class, the **System.Windows.Forms.Application.Exit** method is called. So, use this method in a Windows Forms application, to stop all running message loops and close all appliation windows when executing a start-up [Action](xref:112622).
