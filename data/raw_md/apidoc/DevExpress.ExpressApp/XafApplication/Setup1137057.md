  - id: applicationName
    type: System.String

  - id: objectSpaceProviders
    type: System.Collections.Generic.IList{DevExpress.ExpressApp.IObjectSpaceProvider}
    description: An **IList\<**[](xref:DevExpress.ExpressApp.IObjectSpaceProvider)**>** list used to initialize the [XafApplication.ObjectSpaceProviders](xref:DevExpress.ExpressApp.XafApplication.ObjectSpaceProviders) property and then used to create Object Spaces (see [](xref:DevExpress.ExpressApp.BaseObjectSpace)).
  - id: modulesManager
    type: DevExpress.ExpressApp.ApplicationModulesManager
    description: An **ApplicationModulesManager** object that has a module list, and provides access to the [Controllers](xref:112621) Manager, [View Items](xref:112612) Manager, [Application Model](xref:112579) and model difference storages.
  - id: security
    type: DevExpress.ExpressApp.Security.ISecurityStrategyBase
    description: The **ISecurityStrategyBase** object that is the [Security Strategy](xref:113366) that will be used in the application.
