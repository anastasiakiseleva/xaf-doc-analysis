This method retrieves data from the database and populates the data view's records list if this list is not yet initialized. Otherwise, the cached data records list is used. To remove the cache, use the [XafDataView.Reload](xref:DevExpress.ExpressApp.XafDataView.Reload) method.

**Count** is zero if the data view is disposed via the [XafDataView.Dispose](xref:DevExpress.ExpressApp.XafDataView.Dispose) method.
