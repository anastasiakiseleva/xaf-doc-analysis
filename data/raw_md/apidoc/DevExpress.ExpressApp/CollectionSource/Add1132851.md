syntax:
  content: public override void Add(object obj)
  parameters:
  - id: obj
    type: System.Object
    description: An object that must be added to the Collection Source's collection.

Generally, you do not have to call this method. If you have instantiated an [CollectionSourceBase.ObjectTypeInfo](xref:DevExpress.ExpressApp.CollectionSourceBase.ObjectTypeInfo)**.Type** persistent object, and want it to appear in a List View, save it and call the [View.Refresh](xref:DevExpress.ExpressApp.View.Refresh*) method.
