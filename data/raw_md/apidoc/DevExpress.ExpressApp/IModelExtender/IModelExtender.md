The **IModelExtender** interface exposes a single [IModelExtender.ExtendModelInterfaces](xref:DevExpress.ExpressApp.IModelExtender.ExtendModelInterfaces(DevExpress.ExpressApp.Model.ModelInterfaceExtenders)) method. This method allows you to extend an [Application Model](xref:112580) node. For details on how to extend the Application Model, refer to the [Extend and Customize the Application Model in Code](xref:112810) topic.
