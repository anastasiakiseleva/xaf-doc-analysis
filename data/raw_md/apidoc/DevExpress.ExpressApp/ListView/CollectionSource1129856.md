A Collection Source is used to retrieve a collection of objects of the specified type, from an object space that will be used by a [List View](xref:112611). The **CollectionSource** property allows you to use the members of the current List View's Collection Source.

To learn about Collection Sources, refer to the [](xref:DevExpress.ExpressApp.CollectionSourceBase) class description.

An example is available in the [](xref:112988) article.
