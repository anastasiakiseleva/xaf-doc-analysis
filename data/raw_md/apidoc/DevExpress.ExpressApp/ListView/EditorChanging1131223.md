---
uid: DevExpress.ExpressApp.ListView.EditorChanging
name: EditorChanging
type: Event
summary: Occurs before changing the current View's [ListView.Editor](xref:DevExpress.ExpressApp.ListView.Editor) property value.
syntax:
  content: public event EventHandler EditorChanging
seealso: []
---
Handle this event to perform custom actions before a List View's Editor is changed. To customize the Editor after it has been changed, handle the [ListView.EditorChanged](xref:DevExpress.ExpressApp.ListView.EditorChanged) event.
