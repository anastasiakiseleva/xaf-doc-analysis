A List View uses a [List Editor](xref:113189) to display a collection of objects. The List Editor binds this collection to a particular control. You can use built-in List Editors from the **System Module** and [Additional Modules](xref:118046) or create a custom editor in an [ASP.NET Core Blazor](xref:403258) or [Windows Forms](xref:112659) XAF application.

Use the `Editor` property to access members of the current List View's List Editor (for example, `Name`, `DataSource`, `EditMode`, or `ProcessSelectedItemAction`).
