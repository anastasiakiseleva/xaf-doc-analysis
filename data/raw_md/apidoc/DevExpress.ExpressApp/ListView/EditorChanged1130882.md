name: EditorChanged
type: Event
summary: Occurs after the current List View's [ListView.Editor](xref:DevExpress.ExpressApp.ListView.Editor) property has been changed.
syntax:
  content: public event EventHandler EditorChanged

Handle this event to perform custom actions after a List View's Editor has been changed. For instance, utilize specific options that are only supported in a particular List Editor in this event's handler.
