Use the `Context` property to specify a context for the target View's Window in a WinForms application. This property is in effect when the @DevExpress.ExpressApp.ShowViewParameters.TargetWindow property is set to `NewWindow`, `NewModalWindow`, or `Default` and the current Frame is a nested Frame.
