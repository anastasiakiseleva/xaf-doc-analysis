A View's caption is included in the caption of the Window which contains this View.

By default, this property is set to the value of the **Caption** attribute from the [Application Model](xref:112580) node specified by the [View.Model](xref:DevExpress.ExpressApp.View.Model) property.

When setting a value to this property, the [View.CaptionChanged](xref:DevExpress.ExpressApp.View.CaptionChanged) event is raised.
