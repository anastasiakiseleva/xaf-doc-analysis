Use this method to close a View. Pass `true` if you need to raise the [View.QueryCanClose](xref:DevExpress.ExpressApp.View.QueryCanClose) event. The `Close` method does not close the View if the [View.QueryCanClose](xref:DevExpress.ExpressApp.View.QueryCanClose) event handler sets the `CancelEventArgs.Cancel` parameter to `true`.

> [!NOTE]
> [!include[event to prohibit view closing in aspnet core blazor apps](~/templates/event to prohibit view closing in aspnet core blazor apps.md)]

If a View can be closed, the `Close` method raises the [View.Closing](xref:DevExpress.ExpressApp.View.Closing) and [View.Closed](xref:DevExpress.ExpressApp.View.Closed) events.

This method is used internally with the parameter set to `true`.
