name: Control

summary: Returns the current [](xref:DevExpress.ExpressApp.View)'s control.
syntax:
  content: public virtual object Control { get; }
  parameters: []
  return:
    type: System.Object
    description: An object that represents the current View's control.
seealso: []
---
Since the controls which represent a View are arranged via the Layout Control on a Template, the value returned by this property is the Layout Manager's Container. You may need to access this container when implementing a custom [Template](xref:112609). Add the View's control to the Template's View site. To learn how to implement custom Templates, refer to the [How to: Create a Custom WinForms Ribbon Template](xref:112618).
