---
uid: DevExpress.ExpressApp.CollectionSourceBase.GetCount
name: GetCount()
type: Method
summary: Returns the number of objects contained in the Collection Source's [CollectionSourceBase.Collection](xref:DevExpress.ExpressApp.CollectionSourceBase.Collection).
syntax:
  content: public virtual int GetCount()
  return:
    type: System.Int32
    description: The number of objects that are contained in the current Collection Source's collection.
seealso: []
---
Use this property to count the objects in the Collection Source's collection. When deriving from the **CollectionSourceBase** class, override this method to return the Collection Source's count of objects.
