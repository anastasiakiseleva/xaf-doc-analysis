---
uid: DevExpress.ExpressApp.CollectionSourceBase.CollectionReloading
name: CollectionReloading
type: Event
summary: Occurs before the Collection Source's [CollectionSourceBase.Collection](xref:DevExpress.ExpressApp.CollectionSourceBase.Collection) has been reloaded.
syntax:
  content: public event EventHandler CollectionReloading
seealso: []
---
This event is raised by the [CollectionSourceBase.Reload](xref:DevExpress.ExpressApp.CollectionSourceBase.Reload) method. Handle the **CollectionReloading** event to perform the required actions before the Collection Source's collection has been repopulated with data from the database.
