Use the **List** property to access collection items by index.

The [CollectionSourceBase.Collection](xref:DevExpress.ExpressApp.CollectionSourceBase.Collection) property may expose a collection that does not implement or support the **IEnumerable** interface (for example, when [Server](xref:118450) data access mode is enabled). In this case, use the [IObjectSpace.GetObjects](xref:DevExpress.ExpressApp.IObjectSpace.GetObjects*) method instead of the **List** property to obtain the collection and iterate its objects.
