---
uid: DevExpress.ExpressApp.NewWindowTarget.MdiChild
name: MdiChild
type: Field
summary: |-
  If the application displays a tabbed multiple document interface (`UIType=TabbedMDI`), the target [View](xref:112611) is invoked in a new tab of the main [Window](xref:112608).

  If the application displays a standard multiple document interface (`UIType=StandardMDI`), the target [View](xref:112611) is invoked in a new MDI child [Window](xref:112608).
syntax:
  content: MdiChild = 1
seealso: []
