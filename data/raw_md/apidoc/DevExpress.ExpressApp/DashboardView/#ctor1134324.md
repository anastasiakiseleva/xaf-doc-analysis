The constructor is used to create and initialize a new [](xref:DevExpress.ExpressApp.DashboardView) object. It initializes its properties and creates a layout manager. Since the [View.Model](xref:DevExpress.ExpressApp.View.Model) property is not initialized, the [View.Caption](xref:DevExpress.ExpressApp.View.Caption) and [View.Id](xref:DevExpress.ExpressApp.View.Id) properties are not initialized as well. So, this constructor is used when the information from the [Application Model](xref:112580) does not need to be applied.

This method is not designed to be used directly from your code. Dashboard Views are created automatically. If you need to create a custom Dashboard View, use the [XafApplication.CreateDashboardView](xref:DevExpress.ExpressApp.XafApplication.CreateDashboardView(DevExpress.ExpressApp.IObjectSpace,System.String,System.Boolean)) method.
