    type: DevExpress.ExpressApp.IObjectSpace

    type: System.Type
    description: The type of objects that will be retrieved from the database into the custom Collection Source. This parameter value is assigned to the [CreateCustomCollectionSourceEventArgs.ObjectType](xref:DevExpress.ExpressApp.CreateCustomCollectionSourceEventArgs.ObjectType) property.

    type: System.String

    type: DevExpress.ExpressApp.CollectionSourceDataAccessMode
    description: A [](xref:DevExpress.ExpressApp.CollectionSourceDataAccessMode) enumeration value that specifies the mode used to access the Collection Source's [CollectionSourceBase.Collection](xref:DevExpress.ExpressApp.CollectionSourceBase.Collection).
  - id: mode
    type: DevExpress.ExpressApp.CollectionSourceMode
    description: A [](xref:DevExpress.ExpressApp.CollectionSourceMode) enumeration value specifying the mode of operation for the Collection Source.
