The constructor is used to create and initialize a new [](xref:DevExpress.ExpressApp.DetailView) object. It initializes its properties and creates a layout manager. Since the [View.Model](xref:DevExpress.ExpressApp.View.Model) property is not initialized, the [ListView.Caption](xref:DevExpress.ExpressApp.ListView.Caption), [View.AllowEdit](xref:DevExpress.ExpressApp.View.AllowEdit), [View.AllowNew](xref:DevExpress.ExpressApp.View.AllowNew), [View.AllowDelete](xref:DevExpress.ExpressApp.View.AllowDelete) and [View.Id](xref:DevExpress.ExpressApp.View.Id) properties are not initialized as well. So, this constructor is used when the information from the [Application Model](xref:112580) does not need to be applied.

This method is not designed to be used directly from your code. Detail Views are created automatically. If you need to create a custom Detail View, use one of the [XafApplication.CreateDetailView](xref:DevExpress.ExpressApp.XafApplication.CreateDetailView*) method overloads.
