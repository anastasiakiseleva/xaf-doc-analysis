---
uid: DevExpress.ExpressApp.DetailView.DefaultRaiseObjectChangedOnControlValueChanged
name: DefaultRaiseObjectChangedOnControlValueChanged
type: Field
summary: Specifies whether the @DevExpress.ExpressApp.IObjectSpace.ObjectChanged event should occur each time you modify a control value in a DetailView (see the [BC5087](https://supportcenter.devexpress.com/ticket/details/bc5087/core-the-iobjectspace-objectchanged-event-is-not-raised-in-response-to-the-editor-s) breaking change). This option affects all Detail Views in your application.
syntax:
  content: public static bool DefaultRaiseObjectChangedOnControlValueChanged
  return:
    type: System.Boolean
    description: '**true** if the @DevExpress.ExpressApp.IObjectSpace.ObjectChanged event should occur each time you modify a control value in a DetailView; otherwise, **false**.'
seealso: []
---
Blazor applications do not support the `true` field value.
