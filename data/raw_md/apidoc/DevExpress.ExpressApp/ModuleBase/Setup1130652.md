This **Setup** method overload is called before calling the [ModuleBase.CustomizeTypesInfo](xref:DevExpress.ExpressApp.ModuleBase.CustomizeTypesInfo(DevExpress.ExpressApp.DC.ITypesInfo)) method. Override this method, to specify the required settings of the current module. Use the **ApplicationModulesManager** object passed as the _moduleManager_ parameter to get information on the modules that have already been loaded.

By default, this method does nothing.

> [!NOTE]
> This method is called at run time, and when the [Application Model](xref:112580) is being loaded at design time.
