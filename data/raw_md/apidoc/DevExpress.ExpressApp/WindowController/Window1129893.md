By default, all Controllers registered in the [Application Model](xref:112580) are activated when a Window is created. You can specify the Window type for which the current Window Controller can be activated, using the [WindowController.TargetWindowType](xref:DevExpress.ExpressApp.WindowController.TargetWindowType) property.

If you need to prohibit activation of the current Window Controller for a particular Window, override the **OnWindowChanging** method.
