syntax:
  content: public override void Dispose()
seealso: []
---
The **Dispose** method releases all resources allocated by the current [](xref:DevExpress.ExpressApp.NonPersistentObjectSpace), and speeds up system performance. For instance, it disposes the current Object Space's Object Context. Usually, you do not need to use this method, because ObjectSpace objects are disposed of together with disposing container Views.
