A Detail View uses a collection of [View Items](xref:112612) to display the [DetailView.CurrentObject](xref:DevExpress.ExpressApp.DetailView.CurrentObject). The items include:

- Property Editors that display object's properties
- Other View Items added in the [Application Model](xref:112580)

The following code snippet obtains collection of View Items collection:

# [C#](#tab/tabid-csharp)
```csharp
using DevExpress.ExpressApp;
using DevExpress.ExpressApp.Editors;
// ...
public class MyViewController : ViewController<DetailView> {
    protected override void OnActivated() {
        base.OnActivated();
        foreach (ViewItem item in View.Items) {
            // ...
        }
    }
    // ...
}
```
***

To modify the current Composite View's View Item collection, use the following methods:
- [CompositeView.AddItem](xref:DevExpress.ExpressApp.CompositeView.AddItem*)
- [CompositeView.InsertItem](xref:DevExpress.ExpressApp.CompositeView.InsertItem*)
- [CompositeView.RemoveItem](xref:DevExpress.ExpressApp.CompositeView.RemoveItem(System.String))

You can also modify this collection and each View Item individually in the Application Model's [!include[Node_Views_DetailView_Items](~/templates/node_views_detailview_items111383.md)] node.
