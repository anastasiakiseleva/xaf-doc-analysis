Set the generic type argument `T` to the required [View Item](xref:112612) type to get an array of View Items of this type found in the current Composite View's [CompositeView.Items](xref:DevExpress.ExpressApp.CompositeView.Items) collection.

For example, you can get all Static Images of a Composite View. For this purpose, set [](xref:DevExpress.ExpressApp.Editors.StaticImage) as a generic type argument.
