When you need to create an Object Space (for example, to create a [View](xref:112611)), do not create it manually. Instead, use the [XafApplication.CreateObjectSpace](xref:DevExpress.ExpressApp.XafApplication.CreateObjectSpace*) method.

If you cannot access the [](xref:DevExpress.ExpressApp.XafApplication) object, use this constructor.
