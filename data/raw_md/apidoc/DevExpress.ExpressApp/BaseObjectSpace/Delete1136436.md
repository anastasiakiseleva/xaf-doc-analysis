Use this method to delete the objects passed as the _objects_ parameter from the database. This method calls a protected virtual method, **DeleteCore**, which must be overridden in the **BaseObjectSpace** class descendants. Note that the objects cannot be deleted immediately. The [BaseObjectSpace.SetModified](xref:DevExpress.ExpressApp.BaseObjectSpace.SetModified*) method is called so that the deleted objects are marked as modified, and therefore, are deleted from the database during the next changes commit (see [BaseObjectSpace.CommitChanges](xref:DevExpress.ExpressApp.BaseObjectSpace.CommitChanges)).

You can implement a custom delete operation instead of the one performed by the **DeleteCore** method. To do this, subscribe to the [BaseObjectSpace.CustomDeleteObjects](xref:DevExpress.ExpressApp.BaseObjectSpace.CustomDeleteObjects) event and set the event handler's _Handled_ parameter to **true**.
