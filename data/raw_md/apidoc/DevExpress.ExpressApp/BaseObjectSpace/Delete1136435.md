type: Method
summary: Marks the specified persistent object and its aggregated objects as deleted from persistent storage.
syntax:
  content: public void Delete(object obj)
  parameters:
  - id: obj
    type: System.Object
    description: An object that is the persistent object to be deleted.

---
Use this method to delete the object specified as the _obj_ parameter from the database. This method calls a protected virtual method, **DeleteCore**, which must be overridden in the **BaseObjectSpace** class descendants. Note that the object cannot be deleted immediately. The [BaseObjectSpace.SetModified](xref:DevExpress.ExpressApp.BaseObjectSpace.SetModified*) method is called so that the deleted object is marked as modified, and therefore, is deleted from the database during the next changes commit (see [BaseObjectSpace.CommitChanges](xref:DevExpress.ExpressApp.BaseObjectSpace.CommitChanges)).

You can implement a custom delete operation instead of the one performed by the **DeleteCore** method. To do this, subscribe to the [BaseObjectSpace.CustomDeleteObjects](xref:DevExpress.ExpressApp.BaseObjectSpace.CustomDeleteObjects) event and set the event handler's _Handled_ parameter to **true**.
