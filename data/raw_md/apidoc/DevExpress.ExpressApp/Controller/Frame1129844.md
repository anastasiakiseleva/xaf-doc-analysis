All the Controllers that can be used in an application are defined in the [Application Model](xref:112580). When a [](xref:DevExpress.ExpressApp.Frame) is created, an instance of each of these Controllers is created and then registered whithin this Frame. Use the **Frame** property to access the Frame within which the current Controller is registered.
