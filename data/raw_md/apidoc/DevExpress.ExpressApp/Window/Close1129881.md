  content: public virtual bool Close(bool isForceRefresh)
  parameters:
  - id: isForceRefresh
    type: System.Boolean
    description: '**true** if the parent Window must be refreshed; otherwise, **false**.'
  return:
    type: System.Boolean
    description: '**true**, if the Window has been successfully closed; otherwise, **false**.'

Each of the **Window** class descendants provides a different implementation of this method. In the **Window** class, this method does nothing, and returns **false**.
