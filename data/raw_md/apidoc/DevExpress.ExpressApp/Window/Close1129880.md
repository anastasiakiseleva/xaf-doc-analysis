The **Close** method is overridden in [](xref:DevExpress.ExpressApp.Window) class descendants. As a method of the [](xref:DevExpress.ExpressApp.Window) class, it returns **false**. Use this overridden method of the [](xref:DevExpress.ExpressApp.Win.WinWindow) class to close a Window in Windows Forms applications. For example, add a **Cancel** [Action](xref:112622) to a [Window Controller](xref:112621) and call the **Close** method of the Controller's [WindowController.Window](xref:DevExpress.ExpressApp.WindowController.Window) in the Action's [SimpleAction.Execute](xref:DevExpress.ExpressApp.Actions.SimpleAction.Execute) event handler.
