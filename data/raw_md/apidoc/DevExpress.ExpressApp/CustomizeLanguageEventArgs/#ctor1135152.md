  content: public CustomizeLanguageEventArgs(string languageName, string userLanguageName, string preferredLanguageName)
  parameters:
  - id: languageName
    type: System.String
    description: A string containing the language currently used by the application. This parameter value is assigned to the [CustomizeLanguageEventArgs.LanguageName](xref:DevExpress.ExpressApp.CustomizeLanguageEventArgs.LanguageName) property.
  - id: userLanguageName
    type: System.String
    description: Specifies the language which is set in the user's operating system or passed by the Internet browser. This parameter value is assigned to the [CustomizeLanguageEventArgs.UserLanguageName](xref:DevExpress.ExpressApp.CustomizeLanguageEventArgs.UserLanguageName) property.
  - id: preferredLanguageName
    type: System.String
    description: A string specifying the language currently set for the [Application Model](xref:112580)'s [IModelApplication.PreferredLanguage](xref:DevExpress.ExpressApp.Model.IModelApplication.PreferredLanguage) property. This parameter value is assigned to the [CustomizeLanguageEventArgs.PreferredLanguageName](xref:DevExpress.ExpressApp.CustomizeLanguageEventArgs.PreferredLanguageName) property.
