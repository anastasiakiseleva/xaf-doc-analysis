The `CustomShowClonedObjectEventArgs` class declares properties specific to the [CloneObjectViewController.CustomShowClonedObject](xref:DevExpress.ExpressApp.CloneObject.CloneObjectViewController.CustomShowClonedObject) event that you can use to implement the custom code instead of displaying the Detail View with the cloned object.
