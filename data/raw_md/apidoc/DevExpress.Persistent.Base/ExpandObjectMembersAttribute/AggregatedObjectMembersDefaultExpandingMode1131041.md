uid: DevExpress.Persistent.Base.ExpandObjectMembersAttribute.AggregatedObjectMembersDefaultExpandingMode
name: AggregatedObjectMembersDefaultExpandingMode
type: Field
summary: Specifies how an aggregated reference property is displayed in [Views](xref:112611) when it is not decorated with the [](xref:DevExpress.Persistent.Base.ExpandObjectMembersAttribute). By default, the field is set to [ExpandObjectMembers.InDetailView](xref:DevExpress.Persistent.Base.ExpandObjectMembers.InDetailView), which means that an aggregated reference property is represented by several Property Editors in Detail Views, and by an Object Property Editor in List Views.
syntax:
  content: public static ExpandObjectMembers AggregatedObjectMembersDefaultExpandingMode
  return:
    type: DevExpress.Persistent.Base.ExpandObjectMembers
    description: ''
