---
uid: DevExpress.Persistent.Base.IValueManagerBase.Clear
name: Clear()
type: Method
summary: When implemented by a value manager, sets the [IValueManager`1.Value](xref:DevExpress.Persistent.Base.IValueManager`1.Value) property to the default value of the type parameter.
syntax:
  content: void Clear()
seealso: []
---
The default value is `null` for reference types, and zero for value types.
