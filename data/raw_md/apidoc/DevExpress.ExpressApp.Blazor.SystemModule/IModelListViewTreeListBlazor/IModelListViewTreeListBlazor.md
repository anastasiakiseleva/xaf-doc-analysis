This interface is a part of the [Application Model](xref:112580) infrastructure. ASP.NET Core Blazor UI module uses `IModelListViewTreeListBlazor` to extend the @DevExpress.ExpressApp.Model.IModelView interface. You do not need to implement this interface.

This interface is necessary when you use a data source with [flat data](xref:112836#flat-data-structure) objects. For more information, refer to the following topic: [](xref:405312).
