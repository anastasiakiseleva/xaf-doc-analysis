Use the `ActiveTabIndex` property to set the selected tab's index. This value initializes the [DxFormLayoutTabPages.ActiveTabIndex](xref:DevExpress.Blazor.DxFormLayoutTabPages.ActiveTabIndex) property in the ASP.NET Core Blazor Form Layout component.

Note that each tab is a Tabbed Layout Group object. 

You can specify the value of this property in the [Model Editor](xref:112582).
