Use the **CustomCSSClassName** property to apply a custom CSS class to a particular layout item or group. After you apply a custom CSS class name, you can use CSS to customize the layout item.

![xaf ASP.NET Core Blazor custom css class name](~/images/xaf-blazor-custom-css-class-name-example.png)

For more information, refer to the following section: [Customize UI Elements Using CSS](xref:403068#customize-ui-elements-using-css).
