# Business Model Design with Entity Framework Core

XAF WinForms and ASP.NET Core Blazor Server UI support the Entity Framework Core data model. Additionally you can use [Security System for Entity Framework Core](xref:113558) in non-XAF apps. 

Entity Framework Core support in XAF is provided by the following assemblies:
* _DevExpress.ExpressApp.EFCore.v<:xx.x:>.dll_
* _DevExpress.EntityFrameworkCore.Security.v<:xx.x:>.dll_

> [!NOTE]
> [!include[](~/templates/efcoreversion.md)]
> We extensively test our Entity Framework support against Microsoft SQL Server only.
> 
> Please [contact us](https://supportcenter.devexpress.com/ticket/create) if you encounter issues with other database providers.

If you use the [Template Kit](xref:405447) to create an EF Core-based XAF solution, then the kit installs EF Core automatically. For instructions on how to install EF Core manually, refer to the following help topic: [Installing Entity Framework Core](https://learn.microsoft.com/en-us/ef/core/get-started/overview/install).
