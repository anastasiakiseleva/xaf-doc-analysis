# Model Editor: Customize Application Model Visually

The [Application Model](xref:112579) holds metadata that defines the application's UI and behavior. That metadata is first formed from the business model code. Different modules can then apply their own changes. End users can also modify the **Application Model** to adjust their work environment. 

You can examine and edit the **Application Model** on any of those layers. For this purpose, use the **Model Editor**.
