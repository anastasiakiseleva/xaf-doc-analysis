# Data Manipulation and Business Logic

In XAF applications, you do not need to query data directly or via an ORM-specific context. Instead, XAF provides you with a convenient and powerful API independent from particular ORM and DBMS specifics – [Object Space](xref:113707). Topics in this section familiarize you with the Object Space concepts – they describe how you can manipulate data and implement business logic.
