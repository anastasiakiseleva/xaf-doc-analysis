# .NET REST Backend / Web API Service

Follow our tutorial to create an application with the Backend Web API Service.

The [Template Kit](xref:405447) scaffolds a Web API Service with integrated authorization & CRUD operations powered by EF Core and our XPO ORM library. You can use OAuth2, JWT, or custom strategies for authentication alongside tools like Postman or Swagger for API testing.
