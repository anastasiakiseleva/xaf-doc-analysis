# Video Tutorials & Overview

## XAF UI Overview Series with Jose and Joche

These two videos were produced with the help of XAF MVPs Jose Columbie of [Xari](https://xari.io/) and Joche Ojeda of BitFrameworks.

### Part 1. Introduction

In this video, we will rapidly develop a multi-platform project management app for Windows and Web platforms with .NET and Visual Studio. Our LOB app UI created from scratch from a single-codebase (C#) will include data security, validation, color highlighting and a cross-platform business logic - all in under 20 minutes using DevExpress XAF, our .NET application framework.
> [!video https://www.youtube.com/embed/vKivMXbsukc]

### Part 2. Data Access

In this video, we talk about the best ways to access and manipulate data in XAF and how to build highly performant apps. 

> [!video https://www.youtube.com/embed/RUyXX2pJcjM]

## Introduction to XAF by Don Wibier: Carl Franklin's Blazor Train Ep 88 (Webinar Video)

In this webinar, a DevExpress evangelist Don Wibier gives an introduction to XAF, its Solution Wizard options such as data access and security, and builds a Blazor app from an existing database using XPO ORM Data Model Wizard. Don also highlights UI customization using the Model Editor and Security System permissions.

> [!video https://www.youtube.com/embed/h1xDjKmkDMQ]
