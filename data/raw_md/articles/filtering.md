# Data Filtering

Multiple application scenarios require filtering [List Views](xref:112611). For instance, an application's [Security System](xref:113366) may prohibit the display of an entire collection in a List View. You may also need to filter a report's data source when using the [Reports Module](xref:113591). The **XAF** supplies various approaches for filtering. Use this topic to identify the scenario and choose a corresponding solution.
