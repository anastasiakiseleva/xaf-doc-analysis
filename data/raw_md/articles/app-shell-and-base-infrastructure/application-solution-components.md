# Application Solution Components

The first step you perform when you start developing a new application using XAF is creating an application solution using the [Template Kit](xref:405447) as described in the <xref:404144> tutorial. The kit creates a special solution type that contains several projects and service folders. Topics in this section demonstrate the purpose of these projects and folders, and illustrate the basics of extending your application and removing parts that are not required.
