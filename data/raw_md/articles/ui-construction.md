# User Interface and Behavior Customization

UI construction is a primary step in building a business application. The **XAF** automatically generates a UI based on your business model and the [Application Model](xref:112580). You can use the default UI or customize it via an Application Model or in code. Topics in this section provide detailed information on XAF UI elements and the mechanisms of their creation and interaction.
