# Model Editor

The Model Editor allows you to view and customize an [Application Model](xref:112580). This tool is available for a developer at design-time and for an application administrator at runtime. The [Ways to Invoke the Model Editor](xref:113326) topic describes how to invoke the Model Editor in both scenarios. 

Refer to the [Model Editor](xref:112582) section for more information about this tool.
