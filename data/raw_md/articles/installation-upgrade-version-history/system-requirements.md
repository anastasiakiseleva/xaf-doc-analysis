# System Requirements

1. [Visual Studio 2022+](https://visualstudio.microsoft.com/) with **ASP.NET and web development** and **.NET desktop development** workloads.
2. [.NET 8+ SDK](https://dotnet.microsoft.com/en-us/download/dotnet)
3. A [free 30-day trial](https://www.devexpress.com/products/try/) version or a licensed version of [DevExpress Universal Subscription](https://www.devexpress.com/Subscriptions/Universal.xml) is installed on your machine.
4. Any [RDBMS](https://en.wikipedia.org/wiki/Relational_database#RDBMS) supported by ORM of your choice ([EF Core](https://learn.microsoft.com/en-us/ef/core/) or by [XPO](xref:1998)) is installed and accessible from your machine to store application data. Refer to the following documentation topics for lists of available database management systems:
    * [Supported Database Providers in EF Core](xref:404290#change-database-provider-in-an-existing-ef-core-application)
    * [Database Systems Supported by XPO](xref:2114)
