# XAF0017: Incompatible assemblies must not be added to the same project

**Severity: Warning**

Assemblies from different platform-specific applications (WinApplication, WebApplication, BlazorApplication) must not be referenced in the same project. For example, do not reference the `DevExpress.ExpressApp.Dashboards.Blazor.v22.1.dll` and `DevExpress.ExpressApp.Dashboards.Win.v22.1.dll` assemblies in the same project.

## How to Fix

Reference only those packages that are actually used in your project. Remove platform-specific assembly references from the _MySolution\Module_ project.
