# XAF0032: Use the explicit syntax to create an instance of the CriteriaOperator class

**Severity: Warning**

Avoid implicit conversion of strings to @DevExpress.Data.Filtering.CriteriaOperator. Use explicit syntax to create the `CriteriaOperator` instances instead.

For more information, refer to the following topics: 
* @DevExpress.Data.Filtering.CriteriaOperator.FromLambda``1(System.Linq.Expressions.Expression{System.Func{``0,System.Boolean}})
* @DevExpress.Data.Filtering.CriteriaOperator.FromLambda``2(System.Linq.Expressions.Expression{System.Func{``0,``1}}) 

## Examples

### Invalid Code

```csharp
CriteriaOperator criteria = "A > B";
```

### Valid Code

```csharp
CriteriaOperator criteria = CriteriaOperator.FromLambda<MyType>(x => x.A > x.B);
// or
CriteriaOperator criteria = CriteriaOperator.Parse("A > B");
```
