# XAF0034: The Copy to Output Directory property of the Model.xafml file must be set to Copy always or Copy if newer

**Severity: Warning**

XAF requires that the Copy to Output Directory of Model.xafml files is set to `Copy always` or `Copy if newer` in platform-specific (_MySolution.Blazor.Server_, _MySolution.Win_) projects. Violation of this rule may lead to Model Editor and application localization issues.

## How to Fix

1. Right-click the _Model.xafml_ file in the Solution Explorer and select "Properties".

2. Set the **Copy to Output Directory** property to "Copy always".
