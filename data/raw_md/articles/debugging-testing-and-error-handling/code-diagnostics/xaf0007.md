# XAF0007: The Build Action property of the XAFML file must be set to Embedded resource

**Severity: Warning**

XAF requires that the Build Action of XAFML files is set to "Embedded resource" in platform-agnostic (_MySolution.Module_) and platform-specific (_MySolution.Blazor.Server_, _MySolution.Win_, _MySolution.Web_) projects. Violation of this rule may lead to Model Editor and application localization issues.

## How to Fix

1. Right-click the XAFML file in the Solution Explorer and select "Properties".

2. Set the **Build Action** property to "Embedded resource".
