# Unit Tests
You can use the [Moq](https://www.nuget.org/packages/Moq/) and [NUnit](https://www.nuget.org/packages/NUnit/) frameworks to test your XAF application. Unlike the approach with [EasyTest](xref:113211), this one does not require an XAF application or [Object Space](xref:113707) instances and [Types Info Subsystem](xref:113669). This topic lists ways to test different parts of an XAF application.

1. [Test Action's enabled/disabled state based on user permissions](https://supportcenter.devexpress.com/ticket/details/t849007/how-to-unit-test-action-s-enabled-disabled-state-based-on-user-permissions)
2. [Test whether object property changes using Actions were successfully committed](https://supportcenter.devexpress.com/ticket/details/t851879/how-to-unit-test-whether-object-property-changes-via-actions-were-successfully-committed)
3. [Test object queries by criteria and Detail View creation](https://supportcenter.devexpress.com/ticket/details/t852589/how-to-unit-test-object-queries-by-criteria-and-detail-view-creation)
4. [Test event handlers in Controllers](https://supportcenter.devexpress.com/ticket/details/t852605/how-to-unit-test-event-handlers-in-controllers)
5. [Test New Action custom business logic based on parent and nested Views](https://supportcenter.devexpress.com/ticket/details/t853287/how-to-unit-test-new-action-s-custom-business-logic-based-on-parent-and-nested-views)
6. [Test the Action's enabled/disabled state based on target criteria and selection dependency types in List View](https://supportcenter.devexpress.com/ticket/details/t853714/how-to-unit-test-action-s-enabled-disabled-state-based-on-target-criteria-and-selection)
7. [Test the Action's enabled/disabled state based on target criteria and selection dependency types in Detail View](https://supportcenter.devexpress.com/ticket/details/t853927/how-to-unit-test-action-s-enabled-disabled-state-based-on-target-criteria-and-selection)
8. [Test localized strings from CaptionHelper (Approach 1)](https://supportcenter.devexpress.com/ticket/details/t853968/how-to-unit-test-localized-strings-from-captionhelper-approach-1)
9. [Test localized strings from CaptionHelper (Approach 2)](https://supportcenter.devexpress.com/ticket/details/t854049/how-to-unit-test-localized-strings-from-captionhelper-approach-2)
10. [Test custom logic in XAF/XPO business classes](https://supportcenter.devexpress.com/ticket/details/t884471/how-to-unit-test-custom-logic-in-xaf-xpo-business-classes)

Refer to the following article for more information on APIs used in these examples: [How to write lightweight unit tests for XAF Actions, Controllers and other custom UI logic](https://supportcenter.devexpress.com/ticket/details/t848996/how-to-write-lightweight-unit-tests-for-xaf-actions-controllers-and-other-custom-ui-logic).
