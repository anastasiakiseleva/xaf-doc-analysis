XAF requires [Multiple Active Result Sets (MARS)](https://learn.microsoft.com/en-us/dotnet/framework/data/adonet/sql/enabling-multiple-active-result-sets) for EF Core applications connected to Microsoft SQL Server. Do not remove `MultipleActiveResultSets=true;` and do not set it to `false` in the connection string. For databases that do not support MARS, XAF has an internal mechanism to function without MARS. To enable it, include `Persist Security Info=True` in the connection string.
