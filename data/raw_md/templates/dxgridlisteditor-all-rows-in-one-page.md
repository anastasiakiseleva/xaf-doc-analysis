If the current View uses @DevExpress.ExpressApp.Blazor.Editors.DxGridListEditor, you can display all rows on one page. Use one of the following methods: 

- Set the [ShowAllRows](xref:DevExpress.ExpressApp.Blazor.SystemModule.IModelListViewBlazor.ShowAllRows) model option to `True`.
- Set the [ShowAllRows](xref:DevExpress.Blazor.DxGrid.ShowAllRows) property of the grid component to `true`. The following article describes how you can use List View API to access the [](xref:DevExpress.Blazor.DxGrid) component and its properties: [](xref:402154).
