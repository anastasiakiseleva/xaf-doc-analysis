When a [](xref:DevExpress.ExpressApp.CollectionSource) object is being created for a [List View](xref:112611), the **DataAccessMode** property returns that List View's Data Access Mode specified in the [Model Editor](xref:112582) (see [IModelListView.DataAccessMode](xref:DevExpress.ExpressApp.Model.IModelListView.DataAccessMode)) or in code (see [CollectionSourceBase.DataAccessMode](xref:DevExpress.ExpressApp.CollectionSourceBase.DataAccessMode)).
