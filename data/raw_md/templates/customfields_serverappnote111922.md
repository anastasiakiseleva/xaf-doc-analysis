[Middle Tier Security](xref:113439) or [any other application](xref:113709) that creates and configures a @DevExpress.ExpressApp.XafApplication cannot access custom fields added in the client XAF application project (in code or in XAFML), or by customizing the application model at runtime. You should add a custom field in a common module and reference it in all applications where you are going to use it.
