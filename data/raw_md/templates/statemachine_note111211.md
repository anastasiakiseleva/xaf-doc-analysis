In certain scenarios, creating a custom [Controller](xref:112621) with the [](xref:DevExpress.ExpressApp.Actions.SingleChoiceAction) Action or a set of [](xref:DevExpress.ExpressApp.Actions.SimpleAction) Actions can be simpler and more straightforward than defining a coded State Machine. So, if you need to define a static state management process that should not be changed by users, consider this approach first instead of using the State Machine module.
