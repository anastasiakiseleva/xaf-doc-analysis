To display localized values of an [enumeration property](xref:113552) in a dashboard, add a [calculated field](xref:113583) to the data source class. Set this field's [IModelMember.Expression](xref:DevExpress.ExpressApp.Model.IModelMember.Expression) to **GetEnumCaption([Enum Property])**. In the dashboard designer, bind items to this calculated field instead of the enumeration property.
