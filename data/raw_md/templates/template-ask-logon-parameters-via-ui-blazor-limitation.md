Note that this property has no effect on Blazor applications. The XAF Blazor login form is displayed if `ClaimsPrincipal.Identity.IsAuthenticated` is set to `false` or if XAF Security is unable to find an XAF user by this principal. For more information on `ClaimsPrincipal`, refer to the following section - [The server responds with a temporary authentication token](xref:404462#2-the-server-responds-with-a-temporary-authentication-token).
