You cannot share non-persistent objects that implement [](xref:DevExpress.ExpressApp.IObjectSpaceLink) between different Object Spaces. For example, an exception is thrown when you open a Detail View for an object selected in a List View. To address this exception, subscribe to the [NonPersistentObjectSpace.ObjectGetting](xref:DevExpress.ExpressApp.NonPersistentObjectSpace.ObjectGetting) event, create a new object instance, and copy property values from the source object.
