By default, each new **PredefinedReportsUpdater** instance removes predefined reports that are not registered within this instance. If you implement your reports in several modules and need to create multiple **PredefinedReportsUpdater** objects in a single application, set the [PredefinedReportsUpdater.UseMultipleUpdaters](xref:DevExpress.ExpressApp.ReportsV2.PredefinedReportsUpdater.UseMultipleUpdaters) property of each updater to **true**.
