> [!NOTE]
> When multiple Object Space Providers are registered in the [XafApplication.ObjectSpaceProviders](xref:DevExpress.ExpressApp.XafApplication.ObjectSpaceProviders) property, the [ModuleUpdater.UpdateDatabaseAfterUpdateSchema](xref:DevExpress.ExpressApp.Updating.ModuleUpdater.UpdateDatabaseAfterUpdateSchema) method is executed multiple times, once for each registered Provider. In this method, before accessing an object of a particular type, check if the current Object Space supports this type using the [IObjectSpace.CanInstantiate](xref:DevExpress.ExpressApp.IObjectSpace.CanInstantiate(System.Type)) method.
