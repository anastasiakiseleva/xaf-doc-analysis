When you create a new project with DevExpress [CLI templates](xref:404967) or the [Template Kit](xref:405447) and enable Middle-Tier Security, HTTPS is enabled by default with an [ASP.NET Core HTTPS development certificate](https://learn.microsoft.com/en-us/aspnet/core/security/enforcing-ssl#trust-the-aspnet-core-https-development-certificate). This certificate cannot be used in non-development environments. For production, configure a proper TLS certificate for the application. For more information, refer to the documentation: [Configure HTTPS](https://learn.microsoft.com/en-us/aspnet/core/fundamentals/servers/kestrel/endpoints?#configure-https).
