To create a Collection Source for the _collectionSource_ parameter, use the [XafApplication.CreateCollectionSource](xref:DevExpress.ExpressApp.XafApplication.CreateCollectionSource*) method. The created Collection Source can be customized to change objects displayed in the created List View. For example, the List View's records can be filtered through the [CollectionSourceBase.Criteria](xref:DevExpress.ExpressApp.CollectionSourceBase.Criteria) property. With this **CreateListView** overload, the [View.ObjectSpace](xref:DevExpress.ExpressApp.View.ObjectSpace) property is initialized using the [CollectionSourceBase.ObjectSpace](xref:DevExpress.ExpressApp.CollectionSourceBase.ObjectSpace) value.
